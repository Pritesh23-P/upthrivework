## Overview

A Master's student at NUS Singapore approached UpthriveWork with a complex capstone project requirement — a real-time IoT monitoring dashboard for smart campus infrastructure. The system needed to handle live sensor data streams, visualize multiple data points simultaneously, and provide an intuitive admin interface for campus administrators.

The student had a clear vision but lacked the time and technical depth to execute a production-quality solution within their three-week deadline while managing other coursework commitments.

## What We Built

Our full stack team designed and developed a complete IoT monitoring system from the ground up. The architecture was built for performance, scalability, and clean code — ensuring it would withstand academic scrutiny.

The frontend was built with React and Recharts, featuring live-updating graphs, a responsive dashboard layout, and an alert notification system. The backend used Node.js with Express, connected to an MQTT broker for real-time sensor data ingestion and MongoDB for persistent storage.
