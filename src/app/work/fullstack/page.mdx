## Overview

A final year computer science student at MIT approached UpthriveWork Hub with a challenging capstone project requirement — a fully functional, production-ready full stack web application to be submitted within two weeks.

The project required a responsive frontend, a RESTful backend API, a relational database, user authentication, and cloud deployment — all documented to academic standards set by the university.

Our team assigned a senior full stack developer who worked closely with the student to understand the project requirements, preferred tech stack, and submission guidelines provided by the faculty.

The final application was delivered three days ahead of the deadline — fully functional, thoroughly documented, and deployed on a live server ready for demonstration to the academic panel.
