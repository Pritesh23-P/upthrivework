## Overview

A PhD candidate at the University of Melbourne was working on their dissertation focused on predicting customer churn using machine learning. The challenge was not just building an accurate model — it required rigorous methodology, explainability using modern interpretability techniques, and a comprehensive academic write-up suitable for journal publication.

With their dissertation deadline approaching, they reached out to UpthriveWork for expert AI/ML support.

## What We Built

Our ML specialists built a complete end-to-end machine learning pipeline in Python. The pipeline included thorough data preprocessing, feature engineering, model selection experiments across multiple algorithms, and final implementation using XGBoost — which delivered the best performance on the dataset.

SHAP (SHapley Additive exPlanations) values were used to provide full model interpretability, making the predictions explainable to both technical and non-technical audiences. The accompanying research documentation covered the full methodology, results analysis, and academic discussion required for journal submission.
